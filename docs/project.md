# Project Overview

Project: Customizable Dashboard based on Open Food Facts  
Framework: Symfony 8  
Runtime: FrankenPHP (hot reload enabled)  
Environment: Docker only (docker compose up)  
Database: PostgreSQL  
Mail testing: Mailhog  
UI: Turbo + Symfony UX Live Components + SortableJS  
UI Kit: Shadcn UI via Symfony UX  
No API Platform.

Goal:
Build a secure, maintainable, DDD-oriented Symfony 8 application that allows authenticated users to create a customizable dashboard based on Open Food Facts data.

Main constraints:
- No React
- No Vue
- No API Platform
- Fully dockerized
- Production-ready quality (technical test for job)
